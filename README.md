# MEW G-code Generator

Static browser-based generator for relative XY-only G-code patterns used in MEW scaffold experiments.

## Local Use

Open `index.html` directly in a browser, or serve the folder locally:

```sh
python3 -m http.server 8765
```

Then visit `http://127.0.0.1:8765/index.html`.

## GitHub Pages

This repository is intended to be published from the `main` branch root with GitHub Pages.
